# Apache Druid
[Apache Druid](https://druid.apache.org/) is a real-time database to power modern analytics applications.


## Druid connection in java application:
- [Http](https://druid.apache.org/docs/latest/querying/sql.html#http-post)
- [JDBC](https://druid.apache.org/docs/latest/querying/sql.html#jdbc)
- [Druidry dependency](https://github.com/zapr-oss/druidry)
